# Food App

## Installation

```bash
# install dependencies
$ npm install

# initialize db
$ npm run db:init

# run migrations
$ npm run migration:up

# fill the db with data
$ npm run fixture
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Usage

You can test payment features with [Stripe Test Cards](https://stripe.com/docs/testing).
Swagger is also available [here](http://localhost:4000/api).
